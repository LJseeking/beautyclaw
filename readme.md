# BeautyClaw Website

BeautyClaw AI Station 官网项目。  
这是一个面向医美行业的静态品牌官网，用于展示 BeautyClaw / OpenClaw 的核心定位、行业解决方案、产品能力、Skill Store 和商务咨询入口。

## 项目定位

BeautyClaw AI Station 是一个基于 OpenClaw 的医美行业 AI Agent 工作站。  
官网核心表达包括：

- OpenClaw 的革命性
- BeautyClaw 对医美行业的深度适配
- AI Agent 工作站 + Skill Store 的平台形态
- 企业微信扫码咨询作为主要转化入口

## 网站结构

当前网站为静态网站，主要页面包括：

- 首页
- OpenClaw
- 行业解决方案
- 产品能力
- Skill Store
- 联系我们
